# Parking System — Distributed & Secure

Sistema distribuído para a gestão de parques de estacionamento com:
- Módulo TCP para comunicação com os lugares
- API REST (Flask) para gestão centralizada
- Criptografia RSA para comunicações seguras
- Dashboard para visualização do estado do parque

---

## Objetivos do projeto

- Registar entradas e saídas de veículos
- Calcular custos do estacionamento
- Comunicar entre módulos via TCP + REST
- Garantir segurança com criptografia assimétrica
